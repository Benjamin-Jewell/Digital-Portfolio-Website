Module Progression

This page highlights the key areas of each module in the first year of the BSc (Hons) Computing and Digital Technologies course, with work I have produced throughout each module, as well as a reflection of my capabilities and achievements within each module


Description

This module introduced the subject of programming and software engineering. It covered areas of software design and implementation using modern design techniques, languages, and environments. It helped to develop the practical skill and knowledge to be able to design and implement software programs using object orientated programming.

The module touches on the fundamental concepts of programming, including software development methodologies; variables, data types, and type conversions; control flow; data structures; while and for loops; handling exceptions and input validation; and much more. It also detailed an introduction to object orientated programming, explaining the key concepts of classes and inheritance, and how it can be implemented in the C# programming language.

Module Reflection

Programming and Software development was already one of the skills I was confident in before starting this course, and the skills I developed through this module only furthered that confidence. While I hadn’t used the C# programming language before, the fundamental concepts of programming were the same, and could easily be applied to the new language. Now that I have learned C#, and have refreshed my knowledge on the fundamentals of programming and software fundamentals, I feel even more confident in my abilities, and am eager to develop my skills more in the future.
