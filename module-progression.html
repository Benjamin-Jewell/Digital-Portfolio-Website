Module Progression

This page highlights the key areas of each module in the first year of the BSc (Hons) Computing and Digital Technologies course, with work I have produced throughout each module, as well as a reflection of my capabilities and achievements within each module


Description

This module introduced the subject of programming and software engineering. It covered areas of software design and implementation using modern design techniques, languages, and environments. It helped to develop the practical skill and knowledge to be able to design and implement software programs using object orientated programming.

The module touches on the fundamental concepts of programming, including software development methodologies; variables, data types, and type conversions; control flow; data structures; while and for loops; handling exceptions and input validation; and much more. It also detailed an introduction to object orientated programming, explaining the key concepts of classes and inheritance, and how it can be implemented in the C# programming language.

Description

This module’s focus was on the fundamental concepts of cyber security, helping to develop the skills and knowledge required to perform analyses and risk assessments of computer systems, as well as provide recommendations for security measures, mitigations, and procedures.

Aspects taught throughout the module include the CIA Triad, Cyber Threat Actors, Attack Vectors, Mitigation Tools and Techniques, Intrusion Detection and Prevention Systems, and Unified Threat Management Systems. These all contribute towards the skills of understanding and identifying risks, and providing countermeasures to mitigate said risks.

Examples of Work

The first assignment of this module had me use a fictional ransomware attack scenario to explain fundamental concepts of cyber security and create a presentation.



The second assignment required me to perform a comprehensive cybersecurity risk assessment for the same organisation that was in the fictional scenario in assignment one. I also had to perform a demonstration of three mitigation tools, showing how each work to address specific risks.



Module Reflection

Whilst I had touched on some sections of cyber security in the past, it is one of the modules that I had less confidence in than some others. I had previously understood some of the fundamental concepts, however the majority of the content was new to me, and was therefore more of a challenge. This meant that completing this module helped expand my skills further, as I now understand more about cybersecurity, and can therefore apply my newfound skills to designing more secure solutions in other areas.

Description

This module introduced the fundamental concepts of data science and their application to a wide range of industries, developing the skills required to clean, manage and process data to produce information viable for decision making.

The module touched on many aspects of data science, the primary being: knowledge of data life cycles, data structures, databases, the use of AI and Machine Learning in data science, as well as the legal and ethical issues around obtaining, processing, storing, and transferring data in modern data science.

Examples of Work

The first assignment of this module required me to create a presentation covering the fundamental concepts of data science, including: the data lifecycle, fundamentals of data storage, challenges in maintaining data quality, and more.



The second assignment of this module required me to select a domain, find/generate data based on that domain, design database schema, then use SQL to create and populate a database, then show the data using SQL queries. Next, I had to read the SQL database into a Pandas Data Frame, perform statistical analysis on the data frame, clean the data frame, produce visualisations from the data within the data frame, then use a machine learning model to make predictions based on the data.



Module Reflection

At the beginning of the module, I already had a limited understanding of Data Science, this meant that progressing and furthering my skills came easier to me than in some other modules. However, there were still large areas that were new to me, and therefore the module still posed more of a challenge compared to a couple other modules. Because of this, the module developed my confidence in Data Science, and I feel a strong amount of progression compared to when I started the module.
